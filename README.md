# ✨ Overview

Ragora AI is a production-grade Retrieval-Augmented Generation platform engineered for scalable document intelligence, semantic retrieval, and enterprise conversational AI systems.

The platform enables developers and organizations to transform unstructured data into searchable semantic knowledge systems powered by Large Language Models.

Unlike traditional chatbot wrappers, Ragora AI focuses on:

- scalable retrieval infrastructure
- low-latency semantic search
- modular AI orchestration
- enterprise-grade backend systems
- production deployment workflows
- scalable inference architecture
- streaming conversational AI
- cloud-native deployment patterns

# 🚀 Local Development

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/ragora-ai.git

cd ragora-ai
```

---

## Backend Setup

```bash
cd backend

python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux/macOS
source .venv/bin/activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
